# enWord_origin

英単語の接頭辞，語根，接尾辞リストです．英単語の勉強の参考にどうぞ．  
特に重要なものは太字にしてあります．

## 接頭辞
| 接頭辞 | 意味 | 変化 | 補足 |
| :-- | :-- | :-- | :-- |
| ab | 離れて | a |
| **ad** | 〜の方へ，〜の方を | ac, ap, at, ar, al, a | 方向や対象を表す |
| con | 共に |
| de | 完全に, 離れて |
| di | 分離 |
| dis | 〜ない |
| equi | 同じの |
| ex | 外に | s, es |
| per | 完全に |
| pre | 前に |
| se | 〜がない |
| sub | 下に | sum |
