# enWord_origin

英単語の接頭辞，語根，接尾辞リストです．英単語の勉強の参考にどうぞ．  

## 接頭辞
重要な接頭辞，意味，変化は太字にしてある．

| 接頭辞 | 意味 | 変化 |
| :-- | :-- | :-- |
| **ad** | **〜の方へ**，**〜の方を** | **ac**，**ap**，**at**，**ar**，**al**，as，**a** |
| **con** | **共に**，**共同の(に)**，**相互の(に)**，**完全に** | **com**，**co**，**col**，**cor** |
| **de** | **離れて**，**下に**，**完全に**，反対 |
| **ex** | **外に** | **e** |
| **for** | **前に**，**前で** | **fore** |
| **ob** | **〜に向かって** | **oc**，**of**，**og**，**op** |
| **per** | **通して**，**完全に** |
| **pro** | **前に**，**前で** | **pre** |
| **sub** | **下に**，近い | **suc**，**suf**，**sug**，**sup**，**sus** |
| **sur** | **上に**，**超えて** | **super**, sir |
